# Credit-Risk-Model-for-Lending-Against-Bitcoin-Collateral
Credit Risk Model for Lending Against Bitcoin Collateral
Overview

This repository contains the code, data analysis, and supporting material for my MSc dissertation.

The project investigates how banks can assess the risk of lending to companies that hold Bitcoin and use Bitcoin as collateral. The focus is on understanding how Bitcoin market crashes affect both borrower default risk and collateral value.

Research Objectives

The project aims to:

Model borrower probability of default (PD) using a Merton-type structural credit model.
Assess the risk of Bitcoin collateral through haircuts and stress testing.
Estimate expected loss (EL) using PD, LGD, and EAD.
Analyse situations where Bitcoin price falls increase both borrower risk and collateral risk simultaneously.
Estimate an appropriate lending spread based on the level of risk.
Methodology

The project combines:

Structural credit risk modelling (Merton model)
Bitcoin market data analysis
Stress testing
Expected loss modelling
Risk-based lending decisions
